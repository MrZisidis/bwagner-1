Lab Exercise

  1. Open Notepad and create a new document named 6_1_table.html and save it to your CSS folder.

  2. Give the document the title: 6_1_table.

  3. In Notepad create a file named 6_1_table.css and save it to your CSS folder.

  4. Use external styling to format your document like the one shown below.

  5. Use a font size of 2.0em for table's caption.

  6. Use the color midnightblue for the table's header.

  7. Use the colors mintcream and white for the row background colors. To alternate the row colors add the following property/value pairs to your style sheet.

    tr:nth-child(even) {
   background-color: mintcream
}
tr:nth-child(odd) {
   background-color: white
}

  8. Center the table horizontally within the document. To center the table add the following property/value pairs to your table styling:

    margin-left:auto; 
margin-right:auto;
