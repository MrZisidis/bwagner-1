Lab Exercise

  1. Open Notepad and create a new document named 12_1_float.html and save it to your CSS folder.

  2. Give the document the title: 12_1_float.

  3. In Notepad create a file named 12_1_float.css and save it to your CSS folder.

  4. Use external styling to create a document with the layout shown below.

  5. The document should contain 6 div elements.

  6. The four content divs should share the following style property/attributes:

    • font-size = 3.0em

    • text-align = center

    • border = 2px solid black

  7. Use the float, width, and height properties to manipulate the page's layout.

  8. Content divs 2 and 4 need to stretch to occupy the remaining width of the page. To accomplish this use the style property overflow: auto.
