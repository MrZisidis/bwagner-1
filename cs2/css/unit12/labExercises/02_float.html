Lab Exercise

  1. Open Notepad and create a new document named 12_2_float.html and save it to your CSS folder.

  2. Give the document the title: 12_2_float.

  3. In Notepad create a file named 12_2_float.css and save it to your CSS folder.

  4. Use external styling to create a document with the layout shown below.

  5. Create two divs. Set the id of the first div to menu and the id of the second div to content.

  6. The menu div should have the following style properties/attributes:

    • position = fixed

    • left position = 5

    • right position = 5

  7. The content div should have the following style properties/attributes:

    • float = left

    • width = 100%

    • margin-left = 200px

    • padding-left = 10px

    • font-size = 3.0em

    • background = white

    • border-left = 2px solid black

  8. Set the body element's background color to lightyellow. Set the background color of the anchor element's hover state to powderblue.

  9. Add 10 paragraph elements to the content div that contain the phrase "Have a nice day!".

  10. View the page in the browser and then scroll the window. Notice how the menu is fixed and does not scroll.
