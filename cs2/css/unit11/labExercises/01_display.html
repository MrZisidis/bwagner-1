  1. Open Notepad and create a new document named 11_1_display.html and save it to your CSS folder.

  2. Give the document the title: 11_1_display.

  3. In Notepad create a file named 11_1_display.css and save it to your CSS folder.

  4. Use the list <li> tag with the display property to create a horizontal menu like the one shown below.

  5. Add a anchor tag for each list item so that each menu item is a link that opens the designated w3school's tutorial in a new browser window. For example the HTML menu item opens the w3school's HTML tutorial.

  6. When the mouse hovers over a menu item it's background color should change. You can use any colors you like for the menu. In case you forgot how to change the anchor tag's hover state here is a link to the tutorial: Links

  7. To add spacing between the menu items change the anchor tag's padding property.
