Lab Exercise

  1. Open Notepad and create a new document named 7_1_border.html and save it to your CSS folder.

  2. In Notepad create a file named 7_1_border.css and save it to your CSS folder.

  3. Use external styling to format your document like the one shown below.

  4. Add the following html to your document:

    <!DOCTYPE html>
<html>
<head>
   <title>7_1_border</title>
</head>
<body>
<div id="div1">
This is a red solid border.
</div>
<div id="div2">
This is a blue ridged border.
</div>
<div id="div3">
This is a gray grooved border.
</div>
<div id="div4">
This is a black outset border.
</div>
</body>
</html>

  5. Use the following styling properties: border, width, margin, and padding.

  6. Center the fourth div element horizontally within the document. Remember how you centered the table in the previous unit?
