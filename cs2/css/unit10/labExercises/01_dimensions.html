Lab Exercise

  1. Open Notepad and create a new document named 10_1_dimension.html and save it to your CSS folder.

  2. Give the document the title: 10_1_dimension.

  3. In Notepad create a file named 10_1_dimension.css and save it to your CSS folder.

  4. Use external styling to format your document like the one shown below.

  5. The document should contain three div elements. Each of these divs share the following properties:

    • red background color

    • 1.4 em font size

    • 100px bottom margin

    • left and right margins set to auto (This will horizontally center the divs)

  6. Each div should have the following unique width and height properties:

    • First div: width = 600px, height = 80px

    • Second div: width-max = 600px, height = 80px

    • Third div: width-max = 600px, height-max = 80px

  7. Once the document is displayed in the browser decrease the width of the browser's window and watch how the different width and height settings effect the appearance of the divs.
