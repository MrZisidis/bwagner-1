4-1 CSS Links

Chapters

  • CSS Links

Lab Exercise

  1. Open Notepad and create a new document named 4_1_link.html and save it to your CSS folder.

  2. Give the document the title: 4_1_link.

  3. Use the anchor tag, two of the four anchor tag state attributes, and internal styling to create a menu system outlined below.

  4. When the menu is displayed it should include 5 anchor tags positioned horizontally with a background color of black and a text color of white. The font size is 1.5em. Also for better spacing add the following property/value setting to the anchor tag styling: padding: 10px; (We cover padding in another unit).

  5. When the mouse hovers over a link the background color should change to yellow and the text color to black.

  6. When the mouse is clicked on a link the background should change to lightgray and the text color to black.

  7. When a link is selected a new tab should be opened in the browser containing the website of the specified search engine(Bing, Google, etc.).
