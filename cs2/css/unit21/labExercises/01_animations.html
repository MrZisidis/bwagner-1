21-1 CSS3 Animations

Chapters

  • CSS3 Animations

Lab Exercise

  1. Open Notepad and create a new document named 21_1_animations.html and save it to your CSS folder.

  2. In Notepad create a file named 21_1_animations.css and save it to your CSS folder.

  3. Copy the following styling to external style sheet:

    body {
   overflow:hidden;
}

.wrapper {
     position: absolute;
     left: 135px;
     top: 0px;
     width:  270px;
     height: 135px;
     animation: move 6s ease-in-out 0s infinite alternate;
     -webkit-animation: move 6s ease-in-out 0s infinite alternate;
}
img {
    position: absolute;
    left: -135px;
    top: 0px;
}

@keyframes move {
    0%   {left: 135px}
    100% {left: 100%} 
}

  4. Add the following code to your HTML document:

    <!DOCTYPE html>
<html>
<head>
   <link rel="stylesheet" type="text/css" href="21_1_animations.css">
   <title>21_1_animations</title>
</head>
<body>
   <div class="wrapper">
       <img src="spongeBob.png">
   </div>
</body>
</html>

  6. View the document in a browser. The Spongebob image should be animated so that it moves back and forth across the entire width of the window.

  7. Modify the styling so that the image rotates 720 degrees as it moves across the screen from left to right. To do this you will need to add the following keyframe declaration:

    @-webkit-keyframes rotate {
   from {-webkit-transform: rotate(0deg); transform: rotate(0deg);}
   to   {-webkit-transform: rotate(720deg); transform: rotate(720deg);} 
}

    You will also need to update the img tag's styling with the following:

    animation: rotate 6s linear 0s infinite alternate;
-webkit-animation: rotate 6s linear 0s infinite alternate;
