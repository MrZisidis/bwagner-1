StudentInfo

Directions

Write a program that prints the following information on the console window: Student Name, Address, City, State, ZipCode, Phone Number, Classification (Freshman, Sophomore, Junior, Senior), Parent or Guardians Name, Address, State, ZipCode, Home Phone, Work Phone, Employer. (The information can be fictitious) The program should use the same format as the sample run below. This includes the dotted lines and spacing between the student section and the parent section.
