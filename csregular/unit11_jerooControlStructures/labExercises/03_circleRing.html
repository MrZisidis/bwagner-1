CirlceRing

Directions

The Ring Islands each have rows of flowers along each outer shore of the island. Write a program that instructs a jeroo to walk all the way around the island, picking each row of flowers as it goes. The jeroo should start and end its walk at location (0,0). The same program must work for both of the islands shown below with no modifications.

Note: Your code should only contain one while structure(loop).
