Jeroo Methods

User-defined methods in Jeroo are coded in the Jeroo methods section. Refer to the illustration below.

Directions

Rewrite the program HomewardAngel, done previously, so that the Jeroo's behavior is divided into the following 3 Jeroo methods:

  • getFlower - this method moves the Jeroo from its starting position to on top of the flower
  • pickAndToss - this method picks up the flower and tosses it at the net
  • enterHouse - this method moves the Jeroo through the opening in the net and finishes inside the house

Use the following method declarations.

// precondition: none
// postcondition: Jeroo has been position on top of a flower
method getFlower()
{

}

// precondition: Jeroo is positioned on top of a flower
// postcondition: flower has been picked and net has been disabled
method pickAndToss()
{

}

// precondition: none
// postcondition: Jeroo has moved inside house
method enterHouse()
{

}

Use the following code as your main method.

method main()
{ 
   Jeroo angel = new Jeroo();
    
   angel.getFlower(); 
   angel.pickAndToss(); 
   angel.enterHouse();
}
