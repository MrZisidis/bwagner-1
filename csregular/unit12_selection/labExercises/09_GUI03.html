GUI03

Directions

Create a GUI program named GUI03 that displays the message "Don't Mess With Texas" like the sample run below.

  • The frame size should be 500 x 500.
  • Follow the four step process to create a JLabel for each word.
    1. create a JLabel variable (instance variable)
    2. create a JLabel object and store its reference in the JLabel variable
    3. set the attributes for the label (location, size, color, etc.)
    4. add the JLabel to the content pane of the frame

    *Create a JLabel variable for each word.

  • Size of each label should be 200 x 60.
  • Font size should be 72.
