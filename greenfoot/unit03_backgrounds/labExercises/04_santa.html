Santa's Sleigh

Download and Open Scenario

In this assignment you are going to create a Scenario that gives the illusion of Santa's sleigh flying with mountains in the background.

  1. Click on the following link to download the Scenario: Santa Sleigh

  2. Select Open.

  3. Copy the folder Santa Sleigh and paste it in your MyGreenfoot folder.

  4. Launch Greenfoot and select Open from the Scenario menu.

  5. Navigate to your MyGreenfoot folder, select the Santa's Sleigh Scenario, and click Open.

Assignment

The Santa's Sleigh Scenario contains two image files. The file Snow Mountains bk.png is a background image and the file santa sleigh.png is an actor image.

  1. Create a World subclass named MyWorld and select the Snow Mountains bk.png as its background image.

  2. Add code to the MyWorld class that will scroll its background from the right to left.

  3. Create an Actor subclass named Santa and select santa sleigh.png as its image.

  4. Add code to the MyWorld class that will add a Santa object to the world.
