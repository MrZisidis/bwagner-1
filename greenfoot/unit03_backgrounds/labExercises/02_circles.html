Circles Background

Create New Scenario

In this assignment you are going to create a new world that's background consists of 50 random circles with each circle having a random location, with a random color, and a random level of transparency.

  1. Open the Greenfoot Environment.

  2. From the Scenario menu select New.

  3. Navigate to your MyGreenfoot folder and type Circles in the Folder name box.

  4. Click the Create button.

Make a CircleWorld Class

  1. Right-click on the World class and select New subclass...

  2. In the New class name box type CircleWorld and click Ok.

Draw CircleWorld Background

  1. Double-click on the CircleWorld class to view its code.

    2. Add code to the CircleWorld constructor that will draw the random circles.

  2. Add the following import statement. (Needed for the Color class)
    3. import java.awt.*;

  3. Add the following line of code below the super command in the constructor.

    4. GreenfootImage bk = getBackground();   // Step 1: get the background image
  4. Create a for loop that loops 50 times.

    5. The following code should placed inside the for loop.

  5. Create 4 integer variables named red, green, blue, and alpha. (Alpha will be used to assign a transparency level. This variable can be assigned a value in the range 0 to 255. 0 is completely transparent (invisible) and 255 is completely opaque.)

  6. Assign a random number to each of the 3 color variables in the range of 0 to 255 inclusive. Reminder: to create a random number use the Greenfoot class's getRandomNumber method.

  7. Assign the alpha variable a random between 1 and 100 inclusive.

  8. To change the drawing color add the following code.

    9. bk.setColor(new Color(red, green, blue, alpha));

  9. Create 3 integer variables named x, y, and width.

  10. Assign the x variable a random number between 0 and 600 inclusive.

  11. Assign the y variable a random number between 0 and 400 inclusive.

  12. Subtract 150 from the x variable. (x -= 150)

  13. Subtract 100 from the y variable. ( y -= 100)

  14. Assign the width variable a random number between 1 and 400 inclusive.

  15. Add the following code to draw a circle.

    16. bk.fillOval(x,y,width,width);

  16. Compile the program and watch the background change.
