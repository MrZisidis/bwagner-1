Gradient Background

Download and Open Scenario

  1. Click on the following link to download the Scenario: Gradient

  2. Select Open.

  3. Copy the folder Gradient and paste it in your MyGreenfoot folder.

  4. Launch Greenfoot and select Open from the Scenario menu.

  5. Navigate to your MyGreenfoot folder, select the Gradient Scenario, and click Open.

Background Image

In this section you will learn how to add a background image to a world. Follow these steps.

  1. Right click on the MyWorld class and select Set image...

  2. Under Image Categories: select background

  3. Under Library Images: select bluerock.jpg and click Ok.

    4. The bluerock image is a tile image. A tile is a small image that is specifically designed so that multiple tiles can fit together to form a larger image. The Greenfoot environment will automatically duplicate the tile to form an image that fills a world's entire background. If the chosen image's size is greater than or equal to the world's dimensions the image will not be duplicated.

Removing Background Image

To remove a background image follow these steps:

  1. Right click on the MyWorld class and select Set image...

  2. Under Scenario images: select No image then click Ok.

Drawing on a Background Image

In this section you will learn how you can draw on a background image using drawing tools. Follow these steps.

  1. Click on the Help menu and select Greenfoot Documentation.

  2. On the left side of the document select GreenfootImage to view the documentation for the GreenfootImage class. Scroll down to the section Method Summary to view the list of methods the class provides.

  3. Drawing on a background is a two step process. First, get the background image from the world and store its reference in a GreenfootImage variable.

    GreenfootImage bk = getBackground();

    Second, use the GreenfootImage methods to change the drawing color or draw on the image.

    bk.setColor(Color.red);          // change color to red
bk.drawLine(0, 0, 200, 200);     // draw a line
bk.fillOval(50, 50, 300, 300);   // draw a filled circle

    4. The example below demostrates this two step process by filling the background of MyWorld with a gradient.

  4. Add the method gradientFill below to the MyWorld class.

    /*
 *  This method fills the background image with a gray gradient.
 *  @param startRange the starting color intensity of the gradient
 *  @param endRange the ending color intensity of the gradient
 */
public void 
    gradientFill
    (int startRange, int endRange)
{
	// Step 1: request the background image from the world and store its reference
	//         in a GreenfootImage variable
	GreenfootImage bk = getBackground();
	
	double repeat = Math.ceil((double)getHeight() / Math.abs(startRange - endRange));
	int y = 0;
	int c = startRange;
	
	for(int i = 0; i < getHeight() / repeat; i++)
	{
		// Step 2: use the GreenfootImage methods to change colors and draw on the 
		//         background image
		bk.setColor(new Color(c, c, c));
		for(int j = 0; j < repeat; j++)
		{
			bk.drawLine(0, y, getWidth(), y);     // Step 2
			y++;
		}
		if(startRange <= endRange)
		   c++;
		else
		   c--;
	}
}

  5. Add the following code to the MyWorld constructor

    gradientFill(50, 255);
