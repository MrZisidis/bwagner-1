Program Modifications

In this lesson you are going to modify the game by having the platforms move down one unit every execution cycle. When the platforms reach the bottom of the world they are moved to the top of the world so they can again move down through the world. This is the same behavior we used to move stars in unit 3.

The class structure in this scenario has been modified. A new class named Platform has been added and both the TopPlatform class and the BottomPlatform class are now declared as subclasses of this new class.

By making TopPlatform and BottomPlatform subclasses of the Platform class any code placed in the Platform class's act method will effect both TopPlatform objects and BottomPlatform objects. In other words, if I want to make both TopPlatforms and BottomPlatforms move all I have to do is place the code in the Platform class's act method and both TopPlatform objects and BottomPlatform objects will automatically move.

  1. Modify the Platform class's act method so that both top platforms and bottom platforms move down one unit every time the act method is called.

  2. Moving the platforms one unit every time the act method is called causes the platforms to move to fast. One way to slow them down is to create a counter variable that is incremented in the act method each time the act method is called, then include an if statement that will only allow the platforms to move when the counter equals a multiple of 5. An example is shown below:

    if(counter % 5 == 0)  // counter is a multiple of 5
{
   setLocation(getX(), getY() + 1);
}
counter++;    // increment counter

    Include this behavior in your code. Make sure to declare the counter variable as an instance variable in the Platform class.

  3. When a platform reaches the bottom of the world move the platform to the top of the world so that it can continue to move through the world from top to bottom. Here is the pseudocode for this process:

    if getY() is greater than the height of the world then
   use setLocation to move platform to top of world
