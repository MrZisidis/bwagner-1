AlienAttack2

Download and Open Scenario

  1. Click on the following link to download the scenario: AlienAttack2

  2. Select Open.

  3. Copy the folder AlienAttack2 and paste it in your myGreenfoot folder.

  4. Launch Greenfoot and select Open from the Scenario menu.

  5. Navigate to your MyGreenfoot folder, select the AlienAttack2 scenario, and click Open.


In this lesson we update the AlienAttack game implemented in the previous unit by adding explosion animations.

AnimatedActor and Explosion Classes

The AnimatedActor class has been added to the scenario. An Explosion class that is a subclass of AnimatedActor has also been added.

Program Modifications

  1. Add the following code to the Phaser class so that when a phaser hits an alien an explosion animation appears. I will leave it to you figure out where to place the code.

    2. Explosion explosion = new Explosion("explosionSheet.gif", 11, 64, 64);
explosion.setAutoAdvance(true);
explosion.setAdvanceSpeed(2);
getWorld().addObject(explosion, actor.getX(), actor.getY());   // actor is a reference to the alien

  2. Add an explosion animation to the Fort class so that an explosion occurs when the fort is hit by a bomb.
