In this lesson a parallax background is added. Parallax is a special effect where multiple layers of a background are scrolled at different speeds. Another trick you can do to enhance the parallax effect and give your game the illusion of depth is to change the transparency levels of images that make up the background.

Parallax Background

In our FlappyBird game parallax is introduced by adding a ground image to the world that sits on top of the background image. This ground image is not part of the background image, instead it is a sprite(actor) that is added to the world. Since the ground is in the foreground it scrolls a little faster than the background image.

Program Modifications

Add a pause game feature that will pause the game when the user presses the "space" key. To continue the game the user presses the "space" key again. To complete this task do the following:

  1. Add the following variable declaration to the FlappyWorld class.

    static boolean paused = false;

    Notice the use of the keyword static in the front of the variable declaration. The static keyword is used to define a special kind of variable called a class variable. Class variables are different from instance variables in that they belong to the class they are defined whereas instance variables belong to an object and are created when an object is instantiated(created). Each object has is own set of instance variables and these variables belong only to that object, whereas a class contains only one version of a class variable and it can be shared among all the objects of that class. Class variables can also be accessed in other classes by using dot notation similar to the way objects access methods but instead of using (object.method) notation use (class.variable) notation. For example suppose you would like to use the paused variable created in FlappyWorld class in the Flappy class. This can be accomplished by using code like the following:

    if(
    FlappyWorld
    .paused == false)
