Adding Pipes

In this lesson pipes are added to the world that FlappyBird has to fly between. There is a top pipe and a bottom pipe that scroll across the world from right to left at regular intervals. The two pipes are positioned vertically at random with the distance between them defined by a constant in the FlappyWorld class named SPACE_BETWEEN_PIPES.

The pipes are organized into three separate classes: Pipe, TopPipe, and BottomPipe.

TopPipe and BottomPipe are subclasses of Pipe. This means they both "act" like a Pipe since they inherit the Pipe class's act method. The reason for organizing the pipes this way is so that all pipes, regardless if they are a TopPipe or BottomPipe, move in the same way but they can have different images associated with them.

A method named createPipes is defined in the FlappyWorld class. Each time it is called in the FlappyWorld act method it creates both a BottomPipe object and a TopPipe object and adds them to the world just out of view to the right.

FlappyBird act Method

There is some interesting code in the FlappyBird act method. Notice that the Pipe class is an argument(parameter) for the method call getOneIntersectingObject.

This code makes use of one of the benefits of inheritance. Since BottomPipe and TopPipe are subclasses of Pipe this if statement will cause the game to end if Flappy collides with any pipe, regardless if the pipe is a BottomPipe or TopPipe. There is no need to check them for collisions separately.

Program Modifications

Implement the FlappyWorld's act method so that the method createPipes is called at regular intervals resulting in the pipes being spaced 100 units a part. To accomplish this task follow these steps:

  1. Define an integer variable named counter as an instance variable for the FlappyWorld class.

  2. Add the following code to the FlappyWorld act method.

    counter++;
System.out.println(counter);

    Compile and execute the program. A console window should appear that displays counter.

  3. Add the following if statement to the act method below the println statement:

    if(counter == 100)
{
   counter = 0;
}

    Compile and execute the program. Notice that counter is reset to zero each time it reaches 100. We can use this to create pipes every 100 steps.

  4. Add the following line of code inside the if statement.

    5. createPipes();

    Compile and execute the program.

  5. Remove the println statement you added in step 2.

  6. Remove the comment symbols from the addObject method call in the FlappyBird constructor so that the FlappyBird will appear. Compile and execute the program.
