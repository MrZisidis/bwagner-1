FlappyBird8

Download and Open Scenario

  1. Click on the following link to download the Scenario: FlappyBird8

  2. Select Open.

  3. Copy the folder FlappyBird8 and paste it in your MyGreenfoot folder.

  4. Launch Greenfoot and select Open from the Scenario menu.

  5. Navigate to your MyGreenfoot folder, select the FlappyBird8 Scenario, and click Open.


In this lesson a startup screen is added to the game.

StartUpScreen Class

This scenario includes three new classes: StartUpScreen, Title, and StartButton.

The StartUpScreen class is a subclass of World. When the game is started an object of this class is created and its world appears on the screen. The purpose of this class is to provide a screen with information about the game such as which keys to use or the current high score.

Two actors are added to the StartUpScreen class named Title and StartButton. The Title class provides a title graphic for the startup screen. The StartButton actor is used as a button that the user clicks on to exit the startup screen and play the game.

Multiple Worlds

When a scenario contains multiple World subclasses how does it know which one to display first? The answer is you have tell it. To notify the scenario of which class should be displayed first do the following:

  1. In our game we want the StartUpScreen world to appear first so you would right-click on it and select new StartUpScreen(). This process selects StartupScreen as the first world to appear when the scenario is executed.

  2. The Greenfoot environment remembers which world is the startup world so you only have to do this selection process once.

Program Modifications

Add sound to the FlappyBird game. In the FlappyBird8 Scenario folder there is a sound folder that contains three sound files named flap.wav, score.wav, and smack.wav. Use the flap sound when Flappy is flying, use score sound when Flappy passes through a set of pipes, and use smack sound when Flappy collides with a pipe.

To add sound to the game do the following:

  1. From the Help menu select Greenfoot Class Documentation.

  2. Click on the class GreenfootSound to view its API(available methods).

  3. In our game we will be using two of the available methods: the constructor and play.

  4. The constructor creates a GreenfootSound object and assigns it a sound from a file.

    GreenfootSound flapSound = new GreenfootSound("flap.wav");

  5. The play method plays the sound.

    flapSound.play();

  6. Add the three sounds to the game with the behavior described above. Use the score sound in the FlappyWorld class and use the flap sound and the smack sound in the Flappy class.
