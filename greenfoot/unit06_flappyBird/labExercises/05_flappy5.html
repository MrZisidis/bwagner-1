Animate Flappy

In this lesson we animate Flappy by making his wings move up and down (flap) as he flies.

To animate Flappy we are going to use the following four images:

Flappy Class

The code used to animate Flappy can be explained in four steps:

  1. Declare Instance Variables

    Look at the new instance variables added to the Flappy class.

    The four image variables are used to store the new images.

    The frame variable is used to track which image(frame) is currently visible.

    The animationCounter variable is a simple counter that is used to flip the frames at regular intervals.

    The ANIMATION_SPEED constant establishes how often the frames are flipped (frame rate).

  2. Define a Flappy Constructor

    Recall that a constructor is executed when an object is first created. This constructor loads all the image files and stores them in variables so that Flappy can use them. Since the 2nd image and the 4th image are identical the constructor assigns the reference of the 2nd image to the 4th image variable. By doing it this way only 3 images have to be loaded from a file instead of four.

  3. Define animateFlappyBird Method

    The animateFlappyBird method is responsible for animating Flappy by cycling through its 4 images at regular intervals. Study the code and see if you can understand how it works.

  4. Make Call to animateFlappyBird

    To animate Flappy the method animateFlappyBird is called in the act method. Placing this call in the act method will cause Flappy to "flap" his wings continually.
