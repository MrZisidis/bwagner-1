In this lesson we learn how to make an actor shoot by demonstrating how to make a Ketchup Cannon.

MyWorld Class

The MyWorld class adds a Cannon object to the middle of the world.

Cannon Class

The cannon shoots a KetchupBall when the left mouse button is pressed. The cannon moves by tracking the mouse's current location. All of the code for the Cannon is in its act method. Study this code until you understand how it works.

Program Modifications

Modify the KetchupBall class's act method so that when the ball has traveled more than 150 units it is removed from the world and replaced with a Splat object. When the Splat appears on the screen play the bloop sound. To accomplish this do the following:

  1. Create an if statement that becomes true when the distance variable is greater than 150. The rest of the code discussed below should be placed inside this if statement.

  2. Before we delete the KetchupBall from the world we need to store some of its information so that it is not lost when it is deleted. Inorder to replace the KetchupBall with a Splat object the Splat object needs to know where the KetchupBall was located when it was deleted and it also needs a reference to the world the KetchupBall belonged. To do this add the following code to the KetchupBall's act method.

    int x = getX();              // store current x value
int y = getY();              // store current y value
World world = getWorld();    // store reference to world it is in

  3. Add the following code to the act method that will remove the KetchupBall from the world.

    getWorld().removeObject(this);    // this means this object

  4. Create a Splat object named splat.

  5. Add the splat object to the world using the World class's addObject method. The addObject method requires three parameters: the object you want to add to the world and the x and y coordinates of the place you would like the object to appear. Since at this point in the code the KetchupBall no longer exists you cannot use its getWorld() method any longer. That is why the world variable was created. Use it to access the world instead: world.addObject(...). Note: the three dots should be replaced by the three parameters addObject requires.

  6. Add the bloop sound to the world so that the bloop sound is played when the splat object appears.
