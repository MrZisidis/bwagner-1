Greenfoot

Greenfoot is an interactive programming environment that allows you to learn programming in a fun and easy way. Based on the Java programming language Greenfoot can be used to create video games, simulations, and other graphics programs.

Change Java's Security Settings

If you cannot view the sample Greenfoot applets you may need to change Java's security settings to allow applets to be run in your browser. To do this follow these steps:

  1. Navigate to the following folder: C:\Program Files\Java\jre...\bin (Select the lastest version of Java's jre)

  2. Scroll the bin folder until you find the file javacpl

  3. Double-click the javacpl file to launch the Java Control Panel.

  4. Click on the Security tab to select it.

  5. Click the button EditSiteList...

  6. Click the Add button.

  7. Click in the small textbox and type: http://www.greenfoot.org

  8. Click the Add button.

  9. Click the Continue button.

  10. Click OK button.

  11. Close the Control Panel.
