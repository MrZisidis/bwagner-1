Installing Greenfoot

  1. Click on the following link: Greenfoot and select Save File.
  2. Once the file has been downloaded double-click on it to start installation. (This process varies depending upon which browser you are using)
  3. The following window should appear.
  4. In the first textbox type: c:\greenfoot
  5. For the second textbox click on the Browse button and navigate to the location of your Java SDK.
