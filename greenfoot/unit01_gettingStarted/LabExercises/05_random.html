  4. Launch Greenfoot and select Open from the Scenario menu.

  5. Navigate to your MyGreenfoot folder, select the Random Numbers Scenario, and click Open.

Directions

In previous lessons you created random numbers using the Math class's random method. The following code generates a random number between 1 and 100 inclusive.

int num = (int)(Math.random() * 100) + 1;

Greenfoot provides another way to generate random numbers. The following code also generates a random number between 1 and 100 inclusive.

int num = Greenfoot.getRandomNumber(100) + 1;

You can still use the Math class's random method if you prefer.

Smiley Class

Click on the Smiley class to view its code. You will notice that it does not contain any methods. The only thing that makes the Smiley class different from the Actor class is there is an image of a smiley face associated with it. Since the Smiley face does not have any special behavior no additional methods are necessary.

MyWorld class

Click on the MyWorld class to view its code.

The MyWorld class includes an act method. Recall that in Greenfoot act methods are called automatically when the user presses the Run button. Any code placed inside these methods will be executed repeatedly while the program is running.

The code in the act method creates a Smiley object and adds it to the world at a random location. Since this code is inside the act method this process is repeated continually. Notice that the code utilizes the getRandomNumber method to place the Smiley objects at random locations within the boundaries of the world. This is accomplished through the use of the World class's getWidth and getHeight methods. These methods return the dimensions of the world.
