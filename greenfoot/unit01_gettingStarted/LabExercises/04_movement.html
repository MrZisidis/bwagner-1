Tracker Class

Look at the Actor classes section for this scenario.

Notice that the Tracker class is a subclass of Actor. This means that a Tracker object can do everything that an Actor can do plus any additional behavior defined in the Tracker class. Double-click on the Tracker class to view its code. Notice there are two methods defined in the Tracker class both named leaveTrack. A Tracker can move just like an Actor but it also has the added feature of being able to leave tracks as it moves.

BoxBug Class

A BoxBug is a subclass of Tracker. This means that a BoxBug is a Tracker and can do everything a Tracker can do, and since a Tracker is an Actor the BoxBug can also do all of the Actor behaviors as well. So a BoxBug is an Actor and a Tracker.

Double-click on the BoxBug class to view its code.

Notice that the BoxBug class extends the Tracker class. This establishes the inheritance behavior described above.

The idea of inheritance in programming is that one class inherits the behavior of another class then extends this behavior so the new subclass can do more than its parent. The new behavior of a BoxBug, defined in its act method, is to travel in a box or square pattern. And since a BoxBug is a Tracker it can also leave tracks as it moves by calling the leaveTrack method it inherited from Tracker.

BoxBug Object in Action

To see a BoxBug's behavior add a BoxBug object to the world using one of the three techniques described in the previous lesson. Use BoxBug as a model to help you make the program modifications listed below.


Program Modifications

  1. A CircleBug is a Tracker. Add code to the CircleBug's act method so that a CircleBug moves in a circle pattern. It should leave tracks as it moves. You can change the color of tracks and their thickness by providing the color and thickness as parameters to the method. For example, the following code will draw tracks in the color red with a thickness of 3:

    leaveTrack(Color.red, 3);

    This code will draw tracks using a custom color with a thickness of 1:

    leaveTrack(new Color(30, 180, 55), 1);

    Hint: if the distance moved and the degrees turned at each step when multiplied together equal 360 then the Bug will make a complete circle.

  2. A StarBug is a Tracker. Add code to the StarBug's act method so that a StarBug moves in a star pattern. It should leave tracks as it moves.

  3. A SpiralBug is a Tracker. Add code to the SpiralBug's act method so that a SpiralBug moves in a spiral pattern. It should leave tracks as it moves.

    Hint: the amount you move needs to increase each time the act method is called. Define any instance variables you find necessary.
