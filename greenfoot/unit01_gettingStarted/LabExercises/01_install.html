Installing Greenfoot

Installing Greenfoot

  1. Click on the following link: Greenfoot.

  2. Once the file has been downloaded double-click on it to start the installation. (This process will vary depending upon the browser)

    If using Google Chrome:

    1. Click Keep.

    2. Click on the file: Greenfoot-generic-241.jar

    If using Firefox:

    1. Click Save File.

    2. Click Download Arrow.

    3. Click on the file: Greenfoot-generic-241.jar

  3. The following window should appear.

  4. In the first textbox click the Browse and navigate to your Desktop.

    The location should be c:/Users/ID#/Desktop/greenfoot where ID# is your ID number.

  5. For the second textbox click the Browse button and navigate to the location of the Java SDK.

    The location is typically c:/Program Files/Java/jdk...)

  6. Click Install

  7. Greenfoot is installed and ready to use!

  8. Click Done

Creating a Greenfoot Shortcut

  1. Open the greenfoot folder on the C: drive.

  2. Right-click on the file named greenfoot and select Create shortcut.

  3. Right-click on the shortcut file and select copy.

  4. Navigate to your Desktop and paste the shortcut.

  5. You can remove the word Shortcut from the name if you like.

  6. Double-click on the shortcut to launch Greenfoot.
